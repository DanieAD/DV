Este repositorio contiene mi proyecto cuyo objetivo es construir una **plataforma funcional completa**, donde:

- La aplicación `app-sample` se construye y testea en pipelines de Azure DevOps.
- La infraestructura se crea con Terraform y se gestiona desde Terraform Cloud.
- Kubernetes despliega la aplicación, primero en local y luego en AKS.
- Azure Pipelines maneja flujos CI/CD tanto para la app como para la infraestructura.
